## [1.0.0] - 2026-07-16

### Added

- Russian classic and international Battleship rule sets
- Exact constrained fleet enumeration for tractable positions
- Weighted Sequential Importance Sampling for larger state spaces
- Two-ply conditional lookahead and bounded endgame expectimax
- Probability heatmap, recommendation panel, fleet tracking, and undo history
- Cancellable Web Worker analysis with synchronous fallback
- Deterministic engine tests, coverage thresholds, ESLint, typecheck, and GitHub Actions CI
- Algorithm, architecture, contribution, conduct, and security documentation
